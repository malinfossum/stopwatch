# Stopwatch

A simple stopwatch application built with vanilla JavaScript as part of my studies at Get Academy.

---

## Run Locally

Open `index.html` directly in your browser  
or use Live Server in VS Code.

---

## Features

- Start, pause, and reset controls
- Timer logic using `setInterval()` and `clearInterval()`
- Dynamic time updates in the DOM
- Basic application state management

---

## Documentation

This project implements a functional stopwatch with real-time updates and user controls.

The goal was to practice core JavaScript concepts including variables, intervals, DOM manipulation, and managing application state in a small interactive interface.
